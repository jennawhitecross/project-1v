# project-1v
R code for my fourth year project on confidence intervals (CIs) for the volume under the ROC Surface (VUS). The repository includes four files: the CI construction methods, the simulation study, the analysis of the simulation results, and the application to a pancreatic cancer diagnosis. The example data is a part of the dataset introduced by Leichtle et al. [2013].


A. B. Leichtle, U. Ceglarek, P. Weinert, C. T. Nakas, J.-M. Nuoffer, J. Kase, T. Conrad, H. Witzigmann, J. Thiery, and G. M. Fiedler. Pancreatic carcinoma, pancreatitis, and healthy controls: metabolite models in a three-class diagnostic dilemma. Metabolomics, 9:677–687, 2013.
